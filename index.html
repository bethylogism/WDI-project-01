Instructions

Mastermind is a code-breaking game. You win the game when you manage to guess all the colors in the code sequence and when they are all in the correct position.

If you reach the end of the board without guessing correctly, you lose!

I will give you clues as to how close you are to the correct pattern ,which may inform your next guess.

Clues:

Are you a code breaker? I have hidden some coloured balls in a particular sequence. Crack the code to win.
